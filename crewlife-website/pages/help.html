Начало работы

Как зарегистрироваться в системе?

+

Для регистрации в системе:

  1. Нажмите кнопку "Регистрация" на главной странице
  2. Заполните форму с вашими данными
  3. Выберите должность и локацию
  4. Нажмите "Зарегистрироваться"

После регистрации вы автоматически войдете в систему и будете перенаправлены в панель управления.

Как войти в систему?

+

Для входа в систему используйте ваши учетные данные:

  • Логин: ваш табельный номер
  • Пароль: пароль, установленный при регистрации

Если вы забыли пароль, обратитесь к администратору системы.

Какие браузеры поддерживаются?

+

Система поддерживает все современные браузеры:

  • Google Chrome (версия 90+)
  • Mozilla Firefox (версия 88+)
  • Safari (версия 14+)
  • Microsoft Edge (версия 90+)

Заявки

Как создать новую заявку?

+

Для создания новой заявки:

  1. Войдите в панель управления
  2. Нажмите на кнопку с нужным типом заявки в разделе "Быстрые действия"
  3. Заполните форму заявки
  4. Нажмите "Отправить заявку"

После отправки заявка получит уникальный номер и будет передана на рассмотрение.

Как отследить статус заявки?

+

Статус заявки можно отследить несколькими способами:

  • В разделе "Мои заявки" в панели управления
  • Через уведомления в системе
  • По email (если включены email-уведомления)

Статусы заявок: "Новая", "В обработке", "Одобрена", "Отклонена", "Выполнена".

Можно ли отменить заявку?

+

Заявку можно отменить только в статусе "Новая". Для отмены:

  1. Найдите заявку в разделе "Мои заявки"
  2. Нажмите кнопку "Отменить"
  3. Подтвердите отмену

После начала обработки заявка не может быть отменена.

Аккаунт

Как изменить личные данные?

+

Для изменения личных данных:

  1. Перейдите в раздел "Профиль"
  2. Выберите вкладку "Личные данные"
  3. Внесите необходимые изменения
  4. Нажмите "Сохранить изменения"

Некоторые данные (табельный номер) изменить нельзя.

Как изменить пароль?

+

Для изменения пароля:

  1. Перейдите в раздел "Профиль"
  2. Выберите вкладку "Безопасность"
  3. Введите текущий пароль
  4. Введите новый пароль дважды
  5. Нажмите "Изменить пароль"

Пароль должен содержать минимум 6 символов.

Техническая поддержка

Что делать, если страница не загружается?

+

Если страница не загружается, попробуйте:

  1. Обновить страницу (F5 или Ctrl+R)
  2. Очистить кэш браузера
  3. Проверить подключение к интернету
  4. Попробовать другой браузер
  5. Обратиться в техническую поддержку

Как включить уведомления?

+

Для включения уведомлений:

  1. Перейдите в раздел "Профиль"
  2. Выберите вкладку "Уведомления"
  3. Включите нужные типы уведомлений
  4. Нажмите "Сохранить настройки"

Для push-уведомлений разрешите их в настройках браузера.
